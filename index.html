Galaxy Zoo 2

This page hosts the drafts and reduction code for the Galaxy Zoo 2 project. It is currently intended for the exclusive use of the GZ science team. Contact Kyle Willett if you have questions about the project, code, or data availability.

Running and working on the repository

If you have a GitHub account, you should be able to download the code and paper drafts from this site. Use the following command:

$ git clone https://github.com/willettk/galaxyzoo2.git

Or, if you want to make changes to the code/contribute to the paper draft via git (which would be really cool to try), you can fork it:

First, click the "Fork" button on https://github.com/willettk/galaxyzoo2. Next:

$ cd your_directory
$ git clone https://github.com/your_username/galaxyzoo2.git
$ git remote add upstream https://github.com/willettk/galaxyzoo2.git
$ git pull upstream

Make some changes to the draft:

$ vim galaxyzoo2/datapaper/gz2datarelease.tex

Finally, go back to your GitHub page and switch to the branch you've been working on. Click the "Pull Request" button, add a title (eg, "Added reference to 1926 Hubble paper") and hit "Send". If we agree on the changes, I'll accept them and they'll be automatically merged into the upstream repository (ie, the master draft).
