Overlapping galaxy pairs

This page links material from the Galaxy Zoo survey for overlapping-galaxy pairs, useful for studies of dust absorption. This survey of the sample derived from the Zoo 1 and Zoo 2 periods (August 2007 - April 2010) is described in detail by Keel et al. (PASP in press, January 2013; arxiv preprint). The resulting catalog of 1990 galaxy pairs is available as plain text or in a sortable JavaScript version(click on a column heading to sort on that quantity).

The file overlapcatalog.txtcontains the complete catalog of overlapping-galaxy pairs described in the PASP manuscript by W. Keel et al. In plain text ASCII, its layout is as follows:

Field Columns Description
SDSS name 1-24 SDSS coordinate name, when available. Otherwise, 2000 coordinates with the same column justification.
SDSS ID 26-43 18-digit SDSS ObjID from DR7. Blank if pair is not within the SDSS DR7 footprint.
r magnitude 1 45-49 r magnitude of brighter galaxy, from SDSS DR7 if available.
r magnitude 2 51-55 r magnitude of fainter galaxy, from SDSS DR7 if available.
z1 57-62 redshift of first object, if known
obj 64-66 identifier for object with z1 - either directional (NSEW and combinations, ctr), fg/bg for foreground and background, or em/abs for blended emission and absorption components. Blank if second galaxy is much fainter. S, SE have different meanings than in "type" field.
z2 69-74 redshift of second object, when known
obj 76-78 identifier for object with z2, as for z1
type 80-84 type of pair, as outlined in the paper. Types are
B background object has small angular size
E two elliptical or S0 galaxies
F face-on spiral and background E/S0 system
Phi edge-on spiral and background E/S0 system
Q background galaxy edge-on and radial to foreground
R might also be polar ring
S two spirals
SE spiral/elliptical overlap not fitting other categories
X nearly edge-on disks crossing in projection
misc none of the above
dz 86-87 contains dz if pair has known Î” z 0.008, blank otherwise
Cross-IDs 90-120 Other object names from common catalogs, or field designations.

The various pair types are illustrated in these two PNG montages (figures 2a and 2b from the manuscript). There are also PDF files including a single page for each pair, with finding charts, more detailed photometry and identifications, and in some cases more precise redshifts.

SDSSOverlaps00Final.pdf 0-10 hours
SDSSOverlaps10final.pdf 10-13 hours
SDSSOverlaps13final.pdf 13-15 hours
SDSSOverlaps15final.pdf 15-24 hours

There is a similar file of candidate pairs which were rejected for the final list, because of evidence for interaction, other geometric reasons, or for having redshifts the wrong way around for dust backlighting: OverlapRejects.pdf and a matching Javascript catalog for them: http://bama.ua.edu/~ammanning/discard.html

All these PDFs are currently hosted in http://astronomy.ua.edu/keel/observe/PDFcharts
